# Roadmap: Investor Showcase App

## Project Overview

**Core Value**: Deliver a "wow" experience for investors by showcasing a visually stunning, responsive dashboard and a seamless ID generation flow that works instantly with minimal data entry.

---

## Phases

- [ ] **Phase 1: Brand, Layout & Dashboard** - Establish the Neon Mint visual identity and core dashboard shell.
- [ ] **Phase 2: Frictionless Registration Flow** - Build the onboarding experience using existing assets and logos.
- [ ] **Phase 3: Instant ID Generation & Preview** - Showcase the high-fidelity real-time ID generation.
- [ ] **Phase 4: Export & Final Polish** - Finalize high-resolution exports and UI performance polish.

---

## Phase Details

### Phase 1: Brand, Layout & Dashboard
**Goal**: Establish the premium visual identity and core navigation shell.
**Depends on**: Nothing
**Requirements**: UI-01, UI-02, DASH-01, DASH-02
**Success Criteria**:
  1. Application uses Neon Mint (#00FFA3) and Zinc/Deep Slate palette across all components.
  2. Responsive centered layout with sticky header is implemented (No Sidebar).
  3. Dashboard landing view is established with placeholder metric cards.
**Plans**: 2 plans
- [ ] 01-01-PLAN.md — Foundation & Brand Identity
- [ ] 01-02-PLAN.md — Dashboard & Activity

### Phase 2: Frictionless Registration Flow
**Goal**: Implement the user onboarding experience optimized for demonstrations using existing branding.
**Depends on**: Phase 1
**Requirements**: UI-03, REG-01, REG-02, REG-03
**Success Criteria**:
  1. UI uses **existing logos** from the web app registration.
  2. Modern multi-step registration form is fully functional.
  3. "Demo Auto-fill" button instantly populates form fields for rapid walkthroughs.
  4. Relaxed validation allows proceeding without encountering "required field" errors.

### Phase 3: Instant ID Generation & Preview
**Goal**: Showcase the high-fidelity real-time ID generation and visual preview.
**Depends on**: Phase 2
**Requirements**: IDGEN-01, IDGEN-02, IDGEN-04
**Success Criteria**:
  1. Instant Front and Back ID preview is displayed immediately after registration.
  2. Canvas-based ID generation produces high-quality visuals with unique QR codes.
  3. ID design adheres perfectly to the premium Neon Mint branding.

### Phase 4: Export & Final Polish
**Goal**: Finalize high-quality export tools and performance.
**Depends on**: Phase 3
**Requirements**: UI-04, IDGEN-03
**Success Criteria**:
  1. High-resolution ID card export (Image or PDF) is functional.
  2. Transitions between views are smooth and professional using Framer Motion.
  3. Entire application flow verified for performance (LCP < 2.5s) and visual fidelity.
